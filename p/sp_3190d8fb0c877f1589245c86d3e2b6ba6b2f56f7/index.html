Especialistas alertam para riscos de expor dados pessoais em etiquetas de encomendas descartadas

Com o aumento das compras online, dados pessoais impressos em etiquetas de entrega podem ser usados por criminosos para aplicar golpes, alertam especialistas em cibersegurança.

Com o aumento das compras online, dados pessoais impressos em etiquetas de entrega podem ser usados por criminosos para aplicar golpes, alertam especialistas em cibersegurança. Etiquetas de encomendas frequentemente contêm informações como nome completo, endereço, telefone e CPF. Após o descarte das embalagens, esses dados podem circular sem controle e cair nas mãos de golpistas, que utilizam essas informações para criar abordagens convincentes e enganar vítimas.

Para reduzir o risco, foi lançada a campanha “Raspe seus Dados”, que incentiva consumidores a remover, rasgar ou raspar os dados pessoais das etiquetas antes de descartar as embalagens. Algumas embalagens até oferecem cupons escondidos que são revelados após a remoção das informações.

Criminosos utilizam técnicas de engenharia social, como vishing, phishing direcionado e roubo de identidade, baseando-se nas informações das etiquetas para ganhar a confiança das vítimas. Essas abordagens funcionam porque os dados aparentam ser legítimos, diminuindo a desconfiança.

O Brasil está entre os países mais afetados por ataques cibernéticos, com milhões de pessoas tendo dados expostos ou usados em golpes. Medidas simples, como destruir etiquetas, evitar fornecer dados sem confirmação e desconfiar de contatos inesperados, podem reduzir significativamente os riscos.

A proteção dos dados pessoais não termina após a compra online, mas continua até o descarte seguro das embalagens. Pequenos hábitos, como raspar as informações das etiquetas, podem ter grande impacto na segurança digital dos consumidores.
